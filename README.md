# Application de gestion des congés et absences

Projet réalisé par Catherine JULES dans le cadre du Mastère Développement Full Stack – Ynov Aix-en-Provence (2025-2026).

Cette application interne permet la gestion des congés, RTT et demandes d'absence, avec des rôles distincts.
